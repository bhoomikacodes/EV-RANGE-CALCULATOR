# EV Range Calculator

An Electric Vehicle (EV) range calculator built in **C** to solve battery range anxiety. It uses battery stats and efficiency metrics to accurately predict if a destination is safely reachable.

### ✨ Features
* **Dynamic Range Calculation:** Calculates exact range based on current battery percentage and total capacity.
* **Destination Checking:** Compares estimated range against the user's destination distance.
* **Smart Alerts:** Warns the user if the battery is too low and calculates exactly how many kilometers short they will fall.

### Tech Stack
* **Language:** C
* **Concepts:** Variables, I/O (`printf`/`scanf`), Float Math, Conditional Logic (`if/else`).

### How to Run
1. Copy the code from `main.c`.
2. Paste it into any C compiler (like Programiz).
3. Click Run and enter your EV's stats when prompted!
